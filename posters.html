Posters
Here are some zipped poster templates for PRz. Additionally the latest version of the layout file, which controls the look of the poster, is included separately. Note that the first version number for layout.tex and the main tex file should match. Older versions can be found here and some examples here.
  • Zip file including poster template v3
Notes:
  • All necessary files are contained in the zip file and should be put in the same folder as the main tex file, or linked appropriately.
  • Please note that due to some of the packages used this will only work with some latex compilers. It is tested with XeLaTeX+BibTeX.
  • Three text colours are predefined, przred, przblue, and przgreen, along with the colours for backgrounds, frames, etc.
  • Suggestions for improvements are welcome!
New in version 3:
  • Links for location of auxiliary files can be defined
New in version 2:
  • Choose whether the abstract has a title.
  • Choose whether the references have a title.
  • Choose between several (basic) colour schemes:
    • 0: Standard, dark background and blue frames
    • 1: Dark background and light gray frames
    • 2: Light background and blue frames
    More may be added later...
  • From v2.1 the background is completely controlled by the layout file. N.B. new image files contained in the latest zip file are required.
