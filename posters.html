Nicholas Sedlmayr Poster Template
Here are some zipped poster templates for UMCS. Note that the first version number for layout.tex and the main tex file should match.
  • Zip file including poster template v3
Notes:
  • All necessary files are contained in the zip file and should be put in the same folder as the main tex file, or linked appropriately using the command \extraslocation in the preamble.
  • Please note that due to some of the packages used this will only work with some latex compilers. It is tested with XeLaTeX+BibTeX.
  • Three text colours are predefined, umcsred, umcsblue, and umcsgreen, along with the colours for backgrounds, frames, etc.
  • Choose whether the abstract has a title.
  • Choose whether the references have a title.
  • Choose between several (basic) colour schemes:
    • 0: Standard, dark background and blue frames
    • 1: Dark background and light gray frames
    • 2: Light background and blue frames
    More may be added later...
  • Several extra useful mathematical terms are defined in the layout file.
  • Suggestions for improvements are welcome!
